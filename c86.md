# Gitをシュタゲで例えてみる

## 目的

シュタゲを通してGitがわからない人でもGitがわかるようになる

## 内容

- Gitの概要
- どこで使える、このようの場合に使える
- どう使える
- GitHubの使い方
- git add
- git commit
- git branch
- git checkout
- git reset
- git cherry-pick
- git clone
- git pull
- git push

## ターゲット

- シュタゲ好きな人、アニメ好きな人
- Git初心者の人

## 初心者が分からないところ

- Gitの概念(まずは知りたい)
- Gitのコマンド
- どこで使える、このようの場合に使える(why)
- どう使える(how)
- +α Gitを使えるようになったらどんなメリット、変化があるか
- Gitを使うために何を勉強すればいいか(付録的な)
  - 参考文献や技術一覧など
