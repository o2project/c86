`https://www.atlassian.com/ja/software/sourcetree/overview`

- Mac OS X: http://www.backlog.jp/git-guide/intro/intro2_1.html[Gitのインストール【チュートリアル1 Gitの基本】 | サルでもわかるGit入門 〜バージョン管理を使いこなそう〜]
